共有スペースにある個室の使用者台帳アプリです。\
Vue.js と firebase を使った認証および CRUD の練習として作成しました。

[**デモサイト**](https://room-status.netlify.app/)\

## 機能

- ログイン/ログアウト
- 利用者のアカウント発行
- 使用状況のリアルタイム取得
- 個室の新規登録/削除

※デモではアカウント発行は出来ません。

## 使用技術

Vue.js 2/ Vue CLI/ Vuetify/ Firebase Authentication/ Firebase Cloud Firestore/ Netlify
